**Grupo 2 - Ponete el Delantal**

*Integrantes: María Luján Leiva, Noelia Argüello, Alejandra Agüero, Gianluca Petrosino*

Este trabajo consistió en desarrollar un sitio web responsive utilizando HTML5 y CSS3, para la página "Ponete el Delantal". Partiendo desde su web que contaba con problemas de diseño y maquetado, nos encargamos de rediseñar completamente el sitio teniendo en cuenta la coherencia visual de la marca y el correcto uso de las semánticas.

Maquetado en Figma:
https://www.figma.com/design/dGSDT67l8Yy3DTubUllXop/G2---Ponete-el-delantal?node-id=324-880&p=f&t=GSz3pi3yok1qkm6f-0
Nuestro proyecto cuenta con 4 htmls vinculados entre sí, una única hoja de estilos,y 2 carpetas donde se encuentra todo el material, como las fuentes tipografías y las imágenes y videos. Y a su vez 2 html en versión mobile, con 2 resoluciones diferentes. 
 
Cada integrante, a partir del archivo css que hicimos en conjunto, realizó un html utilizando distintas ramas, para el control de versiones y posteriormente integrar los cambios al repositorio principal mediante Pull Requests. Se utilizó Visual Studio Code (VS Code) y las herramientas usadas fueron: HTML5, CSS3, GitHub y Figma. Navegador para pruebas: Google Chrome y Microsoft Edge.
